# Linux 排障笔记

> 排障时先确认端口、进程和启动命令，再决定是否重启或强制结束进程。

| 目标 | 命令 | 备注 |
| --- | --- | --- |
| 查看端口占用 | `lsof -i :3000` | 替换成目标端口 |
| 结束进程 | `kill -9 PID` | 优先尝试正常停止服务 |

## 检查端口

```bash
lsof -i :3000
```

## 结束进程

```bash
kill -9 PID
```
